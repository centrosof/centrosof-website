# CENTROSOF

**Global Full-Stack Software Engineering**

Build what works. Break what's broken.

---

## Stack

- **Framework:** React 19 + TypeScript
- **Build:** Vite 8
- **Styling:** Tailwind CSS v4
- **Animation:** Framer Motion
- **Routing:** React Router v7

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
